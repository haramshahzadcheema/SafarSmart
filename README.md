# SafarSmart — Pakistan Travel Intelligence Platform

A full-stack desktop application for comparing travel costs across Pakistan routes.

## Tech Stack
Java 17 | JavaFX | MySQL | BCrypt | JDBC

## Features
- Route cost comparison — own car vs ride-hail vs public transport
- BCrypt-secured user authentication
- Live OGRA fuel prices (April 2026)
- Haversine GPS estimation for unlisted routes
- Bar and pie chart analytics dashboard
- Vehicle CRUD management
- Trip saving and comparison history
